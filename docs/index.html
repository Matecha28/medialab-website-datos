Proyecto académico: Website de Datos

Este sitio web forma parte del curso de Extracción y Procesamiento de Datos. El objetivo es mostrar cómo un/a estudiante de comunicaciones puede extraer información de una página web, procesarla con Python y publicarla en una web académica usando GitHub.

Pregunta del proyecto

¿Qué temas aparecen con mayor frecuencia en los titulares del Blog oficial de Python?

Fuente de datos

Sitio web: https://blog.python.org/
Fecha de extracción: 19 de enero de 2026

Metodología

  • Scraping de titulares usando Python (requests + BeautifulSoup).
  • Limpieza y procesamiento de texto.
  • Cálculo de frecuencias de palabras.
  • Publicación de datos y resultados en GitHub Pages.
